# Sentiment Analysis Management System

A Flask-based web application for sentiment analysis and review management.

## Features

- User Authentication
- Admin Dashboard
- Sentiment Analysis
- CSV/Excel Upload
- Reports (PDF, CSV, Excel)
- Data Visualization
- Audit Logs
- Notifications
- User Management

## Technologies

- Flask
- SQLAlchemy
- Scikit-Learn
- Pandas
- Bootstrap
- SQLite
